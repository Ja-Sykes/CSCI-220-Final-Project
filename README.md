# CSCI-220-Final-Project
University of Saint Francis Rocket League Esports student (and esports team member) made website for CSCI-220 Final Project

Semi-Full Stack website built using HTML, CSS, JS, EJS, Node, MySQL.

# How to Run / Demo
1. Download the folder with all the code.
2. Navigate to the terminal and get directory into folder
3. Run the commands npm init -y then npm i express ejs mysql2 cors express-session
4. Server should start on http://localhost:3000/

# Please Note!
The local MySQL Database I is not hosted on a cloud system and will not work the website I provide here. So fell free to make your own SQL database!
